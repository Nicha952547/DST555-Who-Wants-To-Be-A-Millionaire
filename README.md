## This project is a set of mini games developed using the C programming language
### Game Modes
1. **Number Guessing:** Players attempt to guess a number randomly generated by the computer (1-99).
2. **High-Low Guessing:** Players must predict whether the randomly generated number will be **Higher** or **Lower** than a chosen number.
